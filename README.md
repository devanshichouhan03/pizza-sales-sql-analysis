# pizza-sales sql analysis
This project simulates a real-world business scenario where SQL is used to support decision-making in the food retail industry.

Project Overview
This project analyzes a pizza sales dataset using SQL to extract key business insights related to revenue, customer behavior, and product performance.

 Tools Used

1) MySQL 
2) SQL (Joins, Aggregations, Subqueries, Window Functions)

 Dataset Tables

1) orders
2) order_details
3) pizzas
4) pizza_types

Analysis Performed

1) Total revenue calculation
2) Most ordered pizza size
3) Top 5 pizzas by quantity
4) Revenue contribution analysis
5) Cumulative revenue over time
6) Category-wise performance

 Skills Demonstrated

1) Data Cleaning
2) Data Aggregation
3) JOIN operations
4) GROUP BY & HAVING
5) Subqueries
